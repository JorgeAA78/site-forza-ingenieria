# Plan de Implementación - Rediseño Web Forza Ingeniería

## Objetivo
Rediseñar el sitio web de "Forza Ingeniería" para transmitir profesionalismo, confianza y claridad, utilizando HTML5, CSS3 y Bootstrap 5. El sitio debe ser responsive, moderno y orientado a la conversión (solicitud de cotizaciones).

## Análisis Inicial (Simulado basado en descripción)
- **Estado Actual**: Se percibe una estructura simple, posiblemente tipo "Linktree" o folleto digital, funcional para redes sociales pero limitada para una presencia corporativa robusta.
- **Puntos Débiles**: Falta de jerarquía de información profunda, navegación limitada, y ausencia de contenido SEO-friendly detallado.
- **Oportunidades**: Estructurar la información en páginas dedicadas, mejorar la credibilidad con secciones de "Proyectos" y "Certificaciones", y optimizar los llamados a la acción (CTAs).

## Propuesta de Diseño
### Estructura de Navegación
1.  **Inicio**: Hero section impactante, resumen de servicios, clientes destacados, CTA principal.
2.  **Empresa**: Historia, misión/visión, equipo, certificaciones.
3.  **Servicios**: Detalle técnico de servicios (Media/Baja tensión, Tableros, Mantenimiento).
4.  **Proyectos**: Galería de obras realizadas con filtros o categorías.
5.  **Contacto**: Formulario, mapa, datos de contacto directos.

### Lineamientos Visuales
- **Paleta de Colores**:
    - **Primario**: Azul Marino Profundo (`#002B5B`) - Transmite confianza, ingeniería, seriedad.
    - **Secundario**: Gris Acero (`#6C757D`) - Para textos secundarios y fondos neutros.
    - **Acento**: Azul Eléctrico (`#00AEEF`) o Naranja Seguridad (`#FF6600`) - Para botones y destacados. *Seleccionaremos Azul Eléctrico para un look más tecnológico/limpio.*
- **Tipografía**:
    - **Títulos**: `Montserrat` (Google Fonts) - Geométrica, moderna, fuerte.
    - **Cuerpo**: `Roboto` (Google Fonts) - Legible, técnica, neutra.

Se utilizará contenido realista orientado a ingeniería eléctrica: "Montaje de subestaciones", "Tableros de distribución", "Mantenimiento preventivo en plantas industriales".

## Verificación
- Revisión de responsividad (Mobile First).
- Validación de navegación entre páginas.
- Chequeo de estilos y consistencia visual.
